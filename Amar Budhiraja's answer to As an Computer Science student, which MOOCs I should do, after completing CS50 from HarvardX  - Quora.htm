      Thanks for the A2A.
      There can be multiple directions that you can take after CS50x. It gives you a very solid background for CS. I will suggestion the following path
      1. Data Structures - Sadly, I don't find any course on Data Structures to be good enough. However, to learn DS I suggest following a standard DS book and implement the problems. ITS REALLY IMPORTANT. DON'T SKIP THIS.
      2. Algorithm Design- Tim Roughgarden's Coursera on coursera.org -Page on coursera.org and Page on coursera.org .
      Now that you are finished with CS basics. You can opt for several tracks within the CS domain. I'll give you two examples - Data Science and Software Engineering.

      Data Science:
      1. Introduction to Data Science: Page on coursera.org by Bill Howe (This course has the same content as offered by Prof Bill Howe in Introduction to Data Science. Literally, the same content.) It will give you a good understanding.
      2. MySQL: MySQL is the most important language when it comes to databases which are a crucial part of Data Science.
      3. Python/ R: For doing data science, you will need more tools and techniques. Although, the list can be extensive but you can do almost anything with either Python or R. For learning Python, I'd suggest Intro to Computer Science & Programming Course. It may seem easy from a algorithmic perspective but it will get you pretty smooth on Python. For R, I would suggest R-Programming by John Hopkins. Remember, these courses are very small and they will only head start your language skills. To improve start using Python and R on a daily basis for your scripting purposes.
      4. Data Mining and Machine Learning Courses: I personally like the specialization on Coursera on Data Mining. Clears a lot of concepts with some hands-on projects - Page on coursera.org.
      5. Once you have taken these courses, you will figure out on your own which course you should take next.

      Software Engineering
      1. Web Development: You can start with the basic web dev course on udacity. It will teach you basics. Web Development Online Course Following this course, you can look into other courses like HTML/CSS, AJAX, Java Script on www.udacity.com.
      2. Mobile Development: There are multiple Android Teaching courses that you can take. My Personal favorite is - http://cs76.tv/2012/spring/ . It will give you an overall picture of mobile dev including iOS and Android. Some Android Courses - Can I know few Android online course available for free?
      3. From here on, you can move to open source contributions or take advance courses as per your developed interests
